# VoltCalc

A clean and efficient tool to calculate electricity consumption and costs based on meter readings and tariff bands.

---

## Requirements

- Nodejs 24
- PNPM

---

## Setup Instructions

- Clone the repo using thecommand below.

  ```bash
  git clone https://github.com/PhilipOyelegbin/voltcalc.git

  cd voltcalc
  ```

- Run the command below to install all dependencies

  ```bash
  pnpm install
  ```

- Run the command below to start the application

  ```bash
  pnpm run dev
  ```

> Once running, visit: http://localhost:5173
